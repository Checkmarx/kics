-   **Query name:** ECS Task Definition Invalid CPU or Memory
-   **Platform:** CloudFormation
-   **Severity:** <span style="color:#edd57e">Low</span>
-   **Category:** Resource Management
-   **CWE:** <a href="https://cwe.mitre.org/data/definitions/697.html" onclick="newWindowOpenerSafe(event, 'https://cwe.mitre.org/data/definitions/697.html')">697</a>
-   **URL:** [Github](https://github.com/Checkmarx/kics/tree/master/assets/queries/cloudFormation/aws/ecs_task_definition_invalid_cpu_or_memory)

### Description
In an ECS Task Definition with the FARGATE launch type, specifying an invalid CPU or Memory value will result in an error.<br>
[Documentation](https://docs.aws.amazon.com/AmazonECS/latest/developerguide/task-cpu-memory-error.html)

### Code samples
#### Code samples with security vulnerabilities

        Ref: ECSTaskDefinition

  taskdefinition:
