### Description
Deleting an Azure Key Vault without purge protection enabled can cause permanent loss of keys, secrets, and certificates, leading to unrecoverable data loss and disruption of dependent services.<br>
[Documentation](https://registry.terraform.io/providers/hashicorp/azurerm/latest/docs/resources/key_vault)
