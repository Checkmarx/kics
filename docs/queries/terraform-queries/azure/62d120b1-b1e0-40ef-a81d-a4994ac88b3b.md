### Description
There should be a 'azurerm_monitor_activity_log_alert' resource configured to capture 'delete network security group' events<br>
[Documentation](https://registry.terraform.io/providers/hashicorp/azurerm/latest/docs/resources/monitor_activity_log_alert)

Tests Not Fround

  name                = "example-activitylogalert"
  resource_group_name = azurerm_resource_group.example.name
  location            = azurerm_resource_group.example.location
  scopes              = [azurerm_resource_group.example.id]
