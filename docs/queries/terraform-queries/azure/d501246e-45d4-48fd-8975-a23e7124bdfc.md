### Description
Without a System Assigned identity, the Redis instance cannot use Entra ID for secure RBAC access, increasing the risk of credential exposure or overly permissive access.<br>
[Documentation](https://registry.terraform.io/providers/hashicorp/azurerm/latest/docs/resources/redis_cache)
