-   **Query id:** 455f2e0c-686d-4fcb-8b5f-3f953f12c43c
-   **Query name:** Seccomp Profile Is Not Configured
-   **Platform:** Terraform
-   **Severity:** <span style="color:#ff7213">Medium</span>
-   **Category:** Insecure Configurations
-   **URL:** [Github](https://github.com/Checkmarx/kics/tree/master/assets/queries/terraform/kubernetes/secoomp_profile_is_not_configured)

### Description
Containers should be configured with a secure Seccomp profile to restrict potentially dangerous syscalls<br>
[Documentation](https://registry.terraform.io/providers/hashicorp/kubernetes/latest/docs/resources/pod#annotations)

### Code samples
