### Description
Legacy networks can have an impact for high network traffic projects and are subject to a single point of contention or failure.<br>
[Documentation](https://registry.terraform.io/providers/hashicorp/google/6.36.1/docs/resources/compute_network)

#### Code samples with security vulnerabilities
