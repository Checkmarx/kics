### Description
Cloud SQL Database Instance should have SSL enabled<br>
[Documentation](https://cloud.google.com/sql/docs/mysql/admin-api/rest/v1beta4/instances)

### Code samples
#### Code samples with security vulnerabilities
```yaml title="Positive test num. 1 - yaml file" hl_lines="5"
resources:
  - name: sql-instance
    type: sqladmin.v1beta4.instance
    properties:
      settings:
        tier: db-custom-1-3840

```
```yaml title="Positive test num. 2 - yaml file" hl_lines="7"
resources:
  - name: sql-instance
    type: sqladmin.v1beta4.instance
    properties:
      settings:
        tier: db-custom-1-3840
        ipConfiguration:
          ipv4Enabled: true

```
```yaml title="Positive test num. 3 - yaml file" hl_lines="9"
resources:
  - name: sql-instance
    type: sqladmin.v1beta4.instance
    properties:
      settings:
        tier: db-custom-1-3840
        ipConfiguration:
          ipv4Enabled: true
          requireSsl: false

```


#### Code samples without security vulnerabilities
```yaml title="Negative test num. 1 - yaml file"
resources:
  - name: sql-instance
    type: sqladmin.v1beta4.instance
    properties:
      settings:
        tier: db-custom-1-3840
        ipConfiguration:
          requireSsl: true

```
