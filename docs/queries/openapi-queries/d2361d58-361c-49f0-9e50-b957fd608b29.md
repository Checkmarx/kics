-   **Query id:** d2361d58-361c-49f0-9e50-b957fd608b29
-   **Query name:** Schema With Both ReadOnly And WriteOnly
-   **Platform:** OpenAPI
-   **Severity:** <span style="color:#5bc0de">Info</span>
-   **Category:** Structure and Semantics
-   **CWE:** <a href="https://cwe.mitre.org/data/definitions/665.html" onclick="newWindowOpenerSafe(event, 'https://cwe.mitre.org/data/definitions/665.html')">665</a>
-   **URL:** [Github](https://github.com/Checkmarx/kics/tree/master/assets/queries/openAPI/3.0/schema_with_both_read_only_and_write_only)

### Description
Schema should not have both 'writeOnly' and 'readOnly' set to true<br>
[Documentation](https://swagger.io/specification/#schema-object)

### Code samples
#### Code samples with security vulnerabilities
