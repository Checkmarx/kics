-   **URL:** [Github](https://github.com/Checkmarx/kics/tree/master/assets/queries/openAPI/3.0/parameter_object_schema_content)

### Description
A Parameter Object must contain either a 'schema' property, or a 'content' property, but not both since they are mutually exclusive<br>
[Documentation](https://swagger.io/specification/#parameter-object)
