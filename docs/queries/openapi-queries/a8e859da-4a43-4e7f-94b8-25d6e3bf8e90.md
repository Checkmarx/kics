-   **Query id:** a8e859da-4a43-4e7f-94b8-25d6e3bf8e90
-   **Query name:** Items Undefined (v3)
-   **Platform:** OpenAPI
-   **Severity:** <span style="color:#5bc0de">Info</span>
-   **Category:** Structure and Semantics
-   **URL:** [Github](https://github.com/Checkmarx/kics/tree/master/assets/queries/openAPI/general/items_undefined)

### Description
Schema/Parameter items should be defined when the schema/parameter is set to an array.<br>
[Documentation](https://swagger.io/specification/#schema-object)
