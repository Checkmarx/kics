    RDS Multi-AZ Deployment Disabled

    • Query id: 2b1d4935-9acf-48a7-8466-10d18bf51a69
    • Query name: RDS Multi-AZ Deployment Disabled
    • Platform: CloudFormation
    • Severity: Medium
    • Category: Availability
    • CWE: 284
    • Risk score: 5.1
    • URL: Github

    Description¶

    AWS RDS Instance should have a multi-az deployment
    Documentation
