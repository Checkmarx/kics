    Auto TLS Set To True

    • Query id: 98ce8b81-7707-4734-aa39-627c6db3d84b
    • Query name: Auto TLS Set To True
    • Platform: Kubernetes
    • Severity: High
    • Category: Secret Management
    • URL: Github

    Description¶

    When using etcd commands, the '--auto-tls' should be set to false
    Documentation

    Code samples¶

    Code samples with security vulnerabilities¶

    Postitive test num. 1 - yaml file
    apiVersion: apps/v1
kind: Deployment
metadata:
  name: app-etcd-deployment
spec:
  selector:
    matchLabels:
      app: app
  replicas: 1
  template:
    metadata:
      labels:
        app: app
        version: v1
    spec:
      serviceAccountName: database
      containers:
      - name: database
        image: gcr.io/google_containers/etcd:v3.2.18
        imagePullPolicy: IfNotPresent
        command: ["etcd"]
        args: ["--auto-tls=true"]
      nodeSelector:
        kubernetes.io/hostname: worker02  
    restartPolicy: OnFailure
