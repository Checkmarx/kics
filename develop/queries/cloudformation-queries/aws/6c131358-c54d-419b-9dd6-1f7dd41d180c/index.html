    ECS Cluster Not Encrypted At Rest

    • Query id: 6c131358-c54d-419b-9dd6-1f7dd41d180c
    • Query name: ECS Cluster Not Encrypted At Rest
    • Platform: CloudFormation
    • Severity: High
    • Category: Encryption
    • CWE: 311
    • URL: Github

    Description¶

    Ensure that AWS ECS clusters are encrypted. Data encryption at rest, prevents unauthorized users from accessing sensitive data on your AWS ECS clusters and associated cache storage systems.
    Documentation
