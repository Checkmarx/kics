    Maximum Length Undefined (v3)

    • Query id: 8c8261c2-19a9-4ef7-ad37-b8bc7bdd4d85
    • Query name: Maximum Length Undefined (v3)
    • Platform: OpenAPI
    • Severity: Low
    • Category: Insecure Configurations
    • CWE: 20
    • URL: Github

    Description¶

    String schema should have 'maxLength' defined.
    Documentation
