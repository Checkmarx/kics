    Beta - TKE Cluster Has Public Access

    • Query id: df6928ed-02f4-421f-9a67-a529860dd7e7
    • Query name: Beta - TKE Cluster Has Public Access
    • Platform: Terraform
    • Severity: Medium
    • Category: Insecure Configurations
    • CWE: Ongoing
    • URL: Github

    Description¶

    TKE Cluster 'public_ip_assigned' should be set to false
