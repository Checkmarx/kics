    KICS Roadmap¶

    These are our upcoming features:

    • Add new BoM resources:
      • AWS Aurora
      • AWS DynamoDB
      • AWS RDS
    • BoM for Serverless Framework
    • Support Windows Containers

    Have an idea? Join the GitHub discussions or contact KICS core team at kics@checkmarx.com
