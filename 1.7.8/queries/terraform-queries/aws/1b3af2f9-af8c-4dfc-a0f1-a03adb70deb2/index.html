    Lambda Function With Privileged Role

    • Query id: 1b3af2f9-af8c-4dfc-a0f1-a03adb70deb2
    • Query name: Lambda Function With Privileged Role
    • Platform: Terraform
    • Severity: High
    • Category: Insecure Configurations
    • URL: Github

    Description¶

    It is not advisable for AWS Lambda Functions to have privileged permissions.
    Documentation

    Code samples¶

    Code samples with security vulnerabilities¶

    Postitive test num. 1 - tf file
    resource "aws_lambda_function" "positivefunction1" {
  filename      = "lambda_function_payload.zip"
  function_name = "lambda_function_name"
  role          = aws_iam_role.positiverole1.arn
  handler       = "exports.test"
  source_code_hash = filebase64sha256("lambda_function_payload.zip")
  runtime = "nodejs12.x"

  tags = {
    Name = "lambda"
  }

  environment = {
    variables = {
      foo = "bar"
    }
  }
}
