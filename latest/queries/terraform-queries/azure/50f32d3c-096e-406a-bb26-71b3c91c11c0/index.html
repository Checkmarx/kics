    Beta - Resource Without Diagnostic Settings

    • Query id: 50f32d3c-096e-406a-bb26-71b3c91c11c0
    • Query name: Beta - Resource Without Diagnostic Settings
    • Platform: Terraform
    • Severity: Medium
    • Category: Observability
    • CWE: 778
    • Risk score: 3.0
    • URL: Github

    Description¶

    Resources that support diagnostic settings should be associated with a 'azurerm_monitor_diagnostic_setting' resource for exporting activity logs
    Documentation
