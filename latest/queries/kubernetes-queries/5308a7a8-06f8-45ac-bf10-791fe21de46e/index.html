    Workload Mounting With Sensitive OS Directory

    • Query id: 5308a7a8-06f8-45ac-bf10-791fe21de46e
    • Query name: Workload Mounting With Sensitive OS Directory
    • Platform: Kubernetes
    • Severity: High
    • Category: Insecure Configurations
    • CWE: 200
    • URL: Github

    Description¶

    Workload is mounting a volume with sensitive OS Directory
